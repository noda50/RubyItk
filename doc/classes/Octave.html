Class Octave
In: Octave/Octave.rb
Parent: Object

interface to call octave facilities.

Usage:

  • basic style 
       Octave.new(){|oct| ...}
  • another basic style 
       oct = Octave.new() ;
   oct.start(){|oct| ...}
  • primitive style 
       oct = Octave.new() ;
   oct.start() ;
   ...
   oct.final() ;
  • oneshot style (invoke octave for each command) 
       oct = Octave.new() ;
   oct.<method>

Note:

  • Array of Numeric in Ruby is considered as column vector 
       [1,2,3] => [1;2;3]
  • Array of an Array of Numeric in Ruby is considerd as row vector 
       [[1,2,3]] => [1,2,3]
